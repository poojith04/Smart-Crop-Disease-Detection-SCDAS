# **AI-Powered Disease Detection**

Deep learning–based rice crop disease classification using TensorFlow and Keras, enabling accurate identification of common rice diseases from uploaded leaf images.

Real-time image analysis with high-accuracy predictions

Support for more than 10 rice diseases, including blast, bacterial blight, and tungro

Automated disease-specific treatment recommendations

# **Multilingual Voice Output**

An accessible text-to-speech advisory system designed for farmers with varying literacy levels.

Voice guidance in English and Telugu

Dual-mode text-to-speech using online (gTTS) and offline (pyttsx3) engines

Audio-based treatment instructions for improved accessibility
